## 🔴 OFFLINE SHIELD - LOCAL SAVE MODE
## အင်တာနက်ပြတ်တောက်မှု အကာအကွယ်

### English
**What You See:**
- 🔴 **RED SHIELD ICON** in top-right corner
- 📶 **"Offline Mode"** indicator
- 🔄 **Sync Queue Badge** shows number (e.g., "3 pending")

**What It Means:**
- WiFi connection is down
- Your orders are **SAFE** - saved to local database
- Orders will sync automatically when WiFi returns
- **DO NOT PANIC** - system is working normally

### Burmese (မြန်မာ)
**သင်မြင်ရမည့်အရာ:**
- 🔴 **အနီရောင် ဒိုင်းလွှားပုံ** သည် ညာဘက်အပေါ်ထောင့်တွင် ရှိမည်
- 📶 **"Offline Mode"** အမှတ်အသား
- 🔄 **Sync Queue Badge** သည် နံပါတ်ပြမည် (ဥပမာ "3 pending")

**အဓိပ္ပာယ်:**

- WiFi ပြန်လာသောအခါ အလိုအလျောက် ထပ်တူပြုမည်
- **စိတ်ပူရန်မလို** - စနစ်သည် ပုံမှန်အလုပ်လုပ်နေသည်

## ⚠️ WHAT TO DO IF...
## အကယ်၍ ဖြစ်ပါက...

### 1. Green Checkmark Doesn't Appear (အစိမ်းရောင် အမှန်ခြစ် မပေါ်ပါက)

**English:**
- Wait 2 seconds
- If still no response, tap "Confirm Order" again
- If problem persists, call Manager

**Burmese:**
- 2 စက္ကန့် စောင့်ပါ
- တုံ့ပြန်မှု မရှိပါက "အော်ဒါအတည်ပြု" ကို ထပ်နှိပ်ပါ
- ပြဿနာ ဆက်ရှိနေပါက မန်နေဂျာကို ခေါ်ပါ

---

### 2. Red Shield Stays for More Than 5 Minutes (အနီရောင်ဒိုင်းလွှား 5 မိနစ်ကျော် ရှိနေပါက)

**English:**
- Check WiFi router (green light should be on)
- Inform Manager
- Continue taking orders normally (they are saved locally)

**Burmese:**
- WiFi router ကို စစ်ဆေးပါ (အစိမ်းရောင်မီး တောက်နေရမည်)
- မန်နေဂျာကို အကြောင်းကြားပါ
- အော်ဒါများကို ပုံမှန်ဆက်ယူပါ (ဒေတာဘေ့စ်တွင် သိမ်းဆည်းထားပြီး)

---

### 3. Sync Queue Badge Shows High Number (>10) (Sync Queue နံပါတ် 10 ကျော်ပါက)

**English:**
- **DO NOT RESTART TABLET**
- Inform Manager immediately
- System will recover automatically
- Orders are safe in database

**Burmese:**
- **Tablet ကို ပြန်မဖွင့်ပါနှင့်**
- မန်နေဂျာကို ချက်ချင်း အကြောင်းကြားပါ
- စနစ်သည် အလိုအလျောက် ပြန်လည်ကောင်းမွန်လာမည်
- အော်ဒါများ ဒေတာဘေ့စ်တွင် လုံခြုံပါသည်
